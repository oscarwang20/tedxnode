What is Node?

Node.js is an open source runtime environment for executing JavaScript outside of a browser. Built on Google Chrome's JavaScript Engine (V8 Engine) by Ryan Dahl in 2009, it is most commonly used to build back-end services.

It can be used to build highly capacity web apps, APIs, and mobile apps as well.

Node Architecture

  • Every browser has a JS engine that can take Javascript code and convert it into code that a machine can understand. Examples: Firefox has Spidermonkey, Chrome has their V8 engine
  • Example: We have the document.getElementById(); and in the browser we have the document/window object. These objects allow us to interact with the environment in which our JS code is running
  • Node also contains a JS engine to run JS but it also has certain unique objects that provide a separate environment for our code than the browser
  • Instead of document.getElementById(); we have:
    • fs.readFile()
    • http.createServer()

Event-driven, Asynchronous, Multithreading

  • The programming languages that we're used to--Python, JS, C, Java, etc--are all synchronous by default.
  • This means that they execute their processes one by one, or each line of code one after another.
  • Node also runs by an event driven, non-blocking architecture, which means that it does not need to finish processing a single request before moving onto another.
  • This runtime environment then allows for faster runtimes overall.
