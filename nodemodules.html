What is a module?

Every file in node is a module and at the center of it all we have the main module (app.js for this demo). Every function and variable defined in a file is scoped to that file or module. Modules and the module system are fundamental parts of how Node.js applications are written and structured. Module functionality is exported so that it can be included in other parts.

Node.js has three types of modules: Built-in modules, Local modules, External modules,

Built-in modules

Built-in modules are distributed with Node.js itself. You load them using require.

Local modules

These are modules you write yourself, they are part of your actual code. Local modules are how to reuse code in your project.

External modules

An external module is installed as a dependency, tracked in your package.json, and located in the node_modules/ directory.
